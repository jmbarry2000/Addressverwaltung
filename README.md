# Adressverwaltung (Android App, .NET MAUI)

Eine einfache lokale Adressverwaltung mit:

✔ Adressen anlegen, bearbeiten, löschen  
✔ Suche nach Name, Ort, PLZ, Firma, Telefon  
✔ Lokale SQLite-Datenbank  
✔ Fotoaufnahme pro Kontakt  
✔ Öffnen der Adresse in Google Maps

## Technologie
- .NET MAUI
- SQLite
- MVVM (CommunityToolkit.Mvvm)

## So läuft’s
1. Projekt klonen  
2. Mit Visual Studio öffnen (Android-Emulator oder Gerät)  
3. Build & Run
